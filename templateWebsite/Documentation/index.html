Installation

This is the simple static HTML template so no need any headech to install it.

Unzip package in an empty directory and you will see template folder. In template folder change content according to yours then upload this folder on server root directory.

This demo will run either on localhost or server

  • If you are using XAMPP, place your theme folder inside htdocs/ directory.
  • If you are using WAMP, place your theme folder inside www/ directory.
  • you are using LIVE SERVER, place your theme folder inside public_html/ directory.
General Information

Software requirements

  • Sublime Text 3 used to edit the .HTML, .JS, .CSS and .SCSS files
  • Koala used as precompiler for SCSS to CSS convert.

Apez comes with a flexible file structure that can be easily used for small to large scope projects.

The following scheme displays the template files structure.

  • Documentation - contains documentation of template cusotmisation and support
  • License - contains License file of template
  • Template - contains the main files of the template. which will be uploaded on server.
    • Admin - Contains Admin Template files
      • assets - contains all third party plugin and library and Icons
      • dist - contains custom stylesheets and scripts. Used for apez template
      • images - contains all images files
      • uploads - contains all uploaded files like images, pdf etc
      • index.html
      • other html files
    • Website Template - Contains Website Template files
      • assets - contains all third party plugin and library and Icons
      • dist - contains custom stylesheets and scripts. Used for apez template
      • images - contains all images files
      • inc - contains php file for contact form
      • uploads - contains all uploaded files like images, pdf etc
      • index.html
      • other html files

JS file Structure

This template has dynamic animations, which have been implemented in JavaScript. The JS file is custom.js.

									
										//*************************************************
										//Sidebar menu  ***********************************
										//*************************************************
										$('body').on('click', '.open-sidebar-right', function () 
										{
											var ele = $(this);
											$('body').css('overflow', 'hidden');
											$('body').append('
');
											setTimeout(function(){ $('.sidebar-right').css('right', '0'); }, 50);
										});

										$('body').on('click', '.sidebar-close, .overlay', function () 
										{
											$('body').css('overflow', 'visible');
											$('.sidebar-right').css('right', '-450px');
											setTimeout(function(){ $('.overlay').remove(); }, 200);
										})

										//*************************************************
										//Left Side menu  *********************************
										//*************************************************
										$('body').on('click', '.menu-close', function () 
										{
											var ele = $(this);
											$('#main-wrapper').addClass('page-menu-small');
											ele.find('i').removeClass('fa-hand-point-left');
											ele.find('i').addClass('fa-hand-point-right');
											ele.removeClass('menu-close');
											ele.addClass('menu-open');
										});

										$('body').on('click', '.menu-open', function () 
										{
											var ele = $(this);
											$('#main-wrapper').removeClass('page-menu-small');
											ele.find('i').removeClass('fa-hand-point-right');
											ele.find('i').addClass('fa-hand-point-left');
											ele.removeClass('menu-open');
											ele.addClass('menu-close');
										});
